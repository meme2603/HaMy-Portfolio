About The Project

家にある食材を活用し、親子で楽しくお弁当作りができるアプリ

忙しい日常の中でも、冷蔵庫の食材を無駄なく使いながら、親子のコミュニケーションを深められる体験を提供することを目的としました。

01. 背景・課題＆解決策案

ユーザーの課題

毎日のお弁当作りでは、献立に悩み、食材をうまく使い切れないという課題があります。

  • 毎日の「お弁当何を作ろう？」という悩み
  • 冷蔵庫の食材を上手に使いたいけど献立が決まらない
  • 親子で一緒に過ごす時間が少ない
  • お弁当を作ることはお母さんの仕事と思われる

解決策の提案

親子で一緒に料理(お弁当作り)をする体験で、一緒に過ごせる

  • 冷蔵庫の食材を登録して“使い切り”提案
  • 5〜15分で作れる時短レシピ
  • 子どもと一緒に楽しめる
02. User Flow

ユーザーフロー (User Flow)

忙しい親子のために、最短3ステップで献立を決定し、調理から記録までスムーズに行える体験を設計しました。

01

食材スキャン

冷蔵庫の食材を選択・スキャン

→
02

AIレシピ提案

AIが最適なレシピを提案・決定

→
03

調理・分担

親子で役割を確認しながら調理

→
04

記録・保存

完成写真をアップしカレンダー保存

03. Wireframes

ワイヤーフレーム (Wireframes)

情報の優先順位を整理し、ユーザーが迷わずに操作できるクリーンなレイアウトを構築しました。

04. Key Features

主要機能 (Key Features)

01

AIレシピ提案機能

冷蔵庫にある食材を選択、またはカメラでスキャンするだけで、AIが家族の好みに合わせた最適なレシピを数秒で提案します。献立に迷う時間を「親子の楽しい時間」に変えます。

02

親子の役割分担表示

「パパ・ママの担当」と「こどもの担当」を分かりやすくアイコンで表示。こどもが安全に、かつ主体的に料理に参加できるようなステップ設計にこだわりました。

03

おべんとう記録カレンダー

作った料理を写真に撮って保存。親子でスタンプやコメントを送り合える機能で、日々の自炊が「かけがえのない思い出」として積み重なっていきます。

05. Visual Archive

全画面一覧 (All Screens)

直感的で温かみのあるデザインを、一貫したデザインシステムに基づいて全ての画面に適用しました。

06. Design System

デザインのポイント (Design Point)

Color Palette

食欲をそそり、親子の温かみを感じさせる暖色系をメインカラーに採用しました。

Accent#FFD93D
Text#111111
Base#F8F9FA

Typography

Heading

親子でつくる、たのしい記録。

Body

AIが冷蔵庫の食材から最適なレシピを提案。忙しい毎日の自炊をサポートします。
